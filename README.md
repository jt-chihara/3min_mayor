# 3分市長 — 3‑Minute Mayor

10×10 のマップに施設を配置し、3分間で人口をできるだけ増やすミニ都市開発ゲームです。Vite + React で実装しています。

## 特徴
- 3分タイマー（ゲーム内時間は月=1秒）
- 需要(DEMAND)と収容(CAPACITY)で人口成長を制約
- 線路🛤 + 駅🚉で需要を大きく底上げ、公園🌳でも上昇
- 住宅🏠/マンション🏢で収容増加（20/200人）
- 月次収支: 税収(人口×1万) − 維持費(線路1/駅4/公園1)
- 資金不足時は自動で借入（所持金がマイナス可、利子なし）
- 開始日: 2025年4月

## 遊び方
1. 右側メニューから施設を選び、盤面のマスをクリックして設置します。
2. 線路と駅を隣接させると駅が「接続済み」となり、需要が大幅に増えます。
3. 住宅/マンションで収容を伸ばし、需要と収容の両面を満たすと人口が伸びます。
4. 3分後に終了。最終人口がスコアです。

## 需要と収容のロジック（簡易）
- 需要: `10 + 公園×6 + 線路×2 + (線路に隣接する駅)×80 + (未接続駅)×8`
- 収容: `住宅×20 + マンション×200`
- 成長ポテンシャル: `min(収容ギャップ, 需要ギャップ)`
  - 需要ギャップはおおむね `demand − floor(people/5)` で見積もり

## 実行方法
```
npm install
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
npm run preview  # ビルド成果物のプレビュー
```

## 主要ファイル

- `src/App.jsx`: ゲーム本体（UI/ロジック）
- `src/styles.css`: スタイル
